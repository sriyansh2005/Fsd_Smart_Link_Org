# Smart Link Organizer

A web-based link management platform to save, organize, and access important links.

## Features

- Save links with URL, title, tags, and optional collection
- Manage custom tags and collections
- Search across links, domains, and tags
- Filter by tag and collection
- Dashboard with quick stats and most-opened links
- Profile page for user details
- Data persistence via `localStorage`

## Run

Open `index.html` directly in a browser.

## Notes

- This implementation is front-end only (no backend auth/database).
- Initial data is seeded for quick preview.
